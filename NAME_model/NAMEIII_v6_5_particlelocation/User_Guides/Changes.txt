31/01/2012

Ticket #63: Add a User Guides directory and installation test suite.

Added the directory User_Guides into the NAME directory structure to store
user guide documentation and a basic test suite for verifying the NAME installation.

This directory contains two subdirectories

 - User_Guides/Docs (intended for the storage of user guide documentation), and
 - User_Guides/Test_Runs (for the installation test suite).

A collection of files and folders have been added for running an installation test
on Linux and Windows systems:

 a) two NAME input files (for a single-site met run and an NWP met run), ksh scripts
    for running NAME and plotting output under Linux, batch files for running NAME
    and plotting output under Windows, and a sample single-site met file,
 b) empty subfolders /Output_SingleSiteMet and /Output_NWPMet as placeholders
    for NAME output files during the installation tests,
 c) empty subfolder /Sample_NWP_Met_Data as placeholder for storing sample NWP met files,
 d) subfolder /Code_Python with sample Python routines for plotting NAME output fields
    (written by Eike Mueller).

Andrew Jones
