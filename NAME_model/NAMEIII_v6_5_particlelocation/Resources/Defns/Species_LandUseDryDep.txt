This file contains 5 species blocks.
1st block: Species other than radionuclides.
           $$ perhaps test-nuclide, iodine-tracer should go here?
2nd block: Radionuclides with surface resistances (or zero deposition velocity for
           infinite surface resistance) but without daughter species, for use when one
           is only interested in the parent species and decay chain modelling is not
           required.
3rd block: Radionuclides with surface resistances (or zero deposition velocity for
           infinite surface resistance).
4th block: The same radionuclides as in block 2, with deposition velocities
           suggested by HPA-RPD.
5th block: Radionuclides arising from the decay of the above radionuclides, with
           deposition velocities suggested by HPA-RPD.

Note if no daughter is given for a radioactive species, this implies that the daughter
is stable.

Where there are more than one possible (unstable) daughter species (with branching
ratio > 0.01), multiple lines have been given (although modelling multiple daughter
species is not currently possible in NAME).

The 'Set of Cloud Gamma Parameters' column (included only in blocks 3-5) is blank -
this needs to be completed to calculate cloud gamma.

For the half lives, daughters and branching ratios see:
ICRP (1983). Radionuclide Transformations: Energy and Intensity of
Emissions, ICRP Publication 38. Annals of the ICRP Vols 11-13. Pergamon
Press, Oxford.

The radionuclide half lives, daughters, and branching ratios have been tested by the
HPA.

Note zero surface resistance = max deposition

Note that IODINE_O = organic iodine; IODINE_E = inorganic iodine.

$$ Note deposition values for H2S have been mapped to SO2.  In reality H2S is less
   soluble than SO2 so it's wet deposition is probably between 2 to 10 times less.

$$ Some isotopes of tellurium decay to isotopes of iodine. In such instances the
   decay products have been input as, for example, IODINE-129 and not IODINE_O-129 nor
   IODINE_E-129. The user must decide which form of Iodine they wish Tellurium to
   decay to or they could model both.

$$ Note surface resistance for ozone was 5.00E+01 (version 5.0). Here we have set dry
deposition to zero (surface resistance blank, deposition velocity = 0) as this is more
appropriate for modelling ozone using the 'plume particles' method. However the
surface resistance should be 5.00E+01 (and deposition velocity blank) for ozone
modelled using 'background particles'.

$$ For the radionuclides in block 2 and 3, the origin and validity of the surface
resistances (or zero deposition velocities) is not known. Many have been set to zero
just so they do deposit. This is certainly the case for the non-noble gases (but not
IODINE-129) added on 17/5/07 at the time that they were added, i.e. for BARIUM-137M,
BISMUTH-214, CARBON-14, HYDROGEN-3, and Lead-214, PLUTONIUM-242, POLONIUM-218,
PRASEODYMIUM-144, RADIUM-226, RUBIDIUM-88, SELENIUM-75, SULPHUR-35, URANIUM-234,
YTTERBIUM-169. The surface resistance for IODINE-129 was set to that for IODINE-131.
Note IODINE_O and IODINE_E have (incorrectly) the same deposition properties in these
blocks.

$$ Mean aerosol diameters (in um) for PM10, PARTICULATE, PM10_MINERAL and PM10_SEA_SALT
are estimates and will need to be revised. A value below 1um does not need to be specific.
