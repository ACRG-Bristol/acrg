This file contains two sets of cloud gamma parameters. Both sets consider the same 16 
radionuclides. The first set considers non-binned photon energies and their associated 
cloud gamma parameters. This dataset is not comprehensive in that for some 
radionuclides, such as I-135, only the top 20 radioactively significant photon 
energies are included. This list can be extended for additional photon energies but 
account must be made in NAME III for 'MaxEnergies' which is currently set to 20 and 
impact on run time. The second set of cloud gamma parameters considers binned photon 
energies and their associated cloud gamma parameters. For each radionuclide 
considered, values of all Linear Attenuation Coefficients -> Adult bone surface doses 
pu air kerma have been included for all 12 photon energies, irrespective of whether 
there is a respective value for photon intensity. It is envisaged that if the user 
wishes to use cloud gamma parameters from the binned photon energies section, only the 
rows of data containing values of photon intensity will be copied to the NAME III 
input file. The user can use non-binned and binned energy related cloud gamma 
parameter data in a single NAME III input file, but not in the same instance for a 
single radionuclide. For example binned data could be used for I-135 and non-binned 
data for Kr-85.  

Binned photon energies and associated photon intensities were taken from:
Eckerman K F, Westfall R J, Ryman J C and Cristy M. (1993) Nuclear decay
data files of the dosimetry research group. Oak Ridge, ORNL/TM-12350

Non-binned photon energies  and associated photon intensities were taken from:
ICRP (1983). Radionuclide Transformations: Energy and Intensity of
Emissions, ICRP Publication 38. Annals of the ICRP Vols 11-13. Pergamon
Press, Oxford.

Linear attenuation coefficients are taken from:
Hubbell J H. (1982) Photon mass attenuation and energy-absorption
coefficients from 1 keV to 20 MeV. Int J Appl Radiat Isot 33 1269-1290

Berger buildup factors are taken from:
Jaeger R G et al (eds). (1968) Engineering compendium on radiation
shielding. Vol 1, Shielding fundamentals and methods. Springer-Verlag

Air kerma per unit fluence and dose per unit air kerma values are taken from:
ICRP. (1996) Conversion Coefficients for use in Radiological Protection
against External Radiation. ICRP Publication 74. Ann. ICRP, Vol 26, No
3/4
